# 要件

ビンゴゲームカードを自動生成する

- 乱数を用いて，5x5マスのビンゴカードを生成する．
- 生成したカードを表示する．
- 画面上でのクリックまたはタッチを検知できる．

## 技術仕様

- 単独で動作するアプリ（Webアプリも可）とする．
- ゲーム開発環境（Unityなど）は利用不可とする．

### 動作プラットフォーム

macOS, Windows, Linux（Electron.jsのサポートに準ずる）

### 開発プラットフォーム

macOS, WebStorm

### 開発言語

Typescript

### 利用ライブラリ

Electron, Vite, React, Semantic UI
